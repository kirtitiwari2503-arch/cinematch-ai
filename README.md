# 🎥 CineMatch AI

## About

CineMatch AI is a movie discovery application built using Python and TMDB (The Movie Database) API. It allows users to search for movies and instantly view information such as movie posters and release dates through a simple and interactive interface.

The project was developed to learn API integration, Python application development, and deployment of web applications.

### File Description

**app.py**

* Main application interface
* Handles user interaction and displays movie information

**tmdb_helper.py**

* Connects to TMDB API
* Fetches movie details, posters, and release dates

**requirements.txt**

* Contains required Python dependencies

---

## ⚙️ How It Works

1. User enters a movie name.
2. The application sends a request to the TMDB API.
3. TMDB returns movie information.
4. The application processes the response.
5. Movie details such as poster and release date are displayed.
